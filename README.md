# Bogacki Fizjoterapia

A production website for a physiotherapy clinic built with Next.js, TypeScript, and Supabase. Focused on performance, accessibility, content management, and delivering a modern healthcare web experience.

[🌐 Live Website](https://bogackifizjoterapia.pl) | [📂 GitHub Repository](https://github.com/mstobrawa/bogacki-fizjoterapia)

Next.js 15 • Production Website • Lightweight CMS

---

## 🛠️ Tech Stack

**Frontend:**
Next.js 15 (App Router), TypeScript, Tailwind CSS

**Backend & Database:**
Supabase (PostgreSQL)

**Authentication:**
Supabase Auth

**Storage:**
Supabase Storage

**Email Integration:**
Resend

**Deployment:**
Vercel

---

## Why this project?

This project was created as a production website for a physiotherapy clinic with a strong focus on usability, performance, and easy content management.

The goal was not only to deliver a modern visual experience but also to provide a lightweight administration system allowing content updates without touching code.

## 🚀 Getting Started

### Prerequisites

Make sure you have:

* Node.js (v18 or higher)
* npm installed

---

### Environment Setup

Create `.env.local` based on `.env.example`

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
RESEND_API_KEY=
ADMIN_EMAILS=
CONTACT_EMAIL=
```

Environment variables are not included in this repository.

---

### Run Locally

Clone repository:

```bash
git clone https://github.com/mstobrawa/bogacki-fizjoterapia.git
```

Install dependencies:

```bash
cd bogackifizjo
npm install
```

Start development server:

```bash
npm run dev
```

Open:

```txt
http://localhost:3000
```
